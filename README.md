## Overview

This project presents a Business Intelligence dashboard developed in Power BI using the Brazilian E-Commerce Public Dataset by Olist.

The dashboard provides insights into customer behavior, seller performance, product categories, sales trends, and geographic distribution of e-commerce activities across Brazil between 2016 and 2018.

## Dataset

Source:
Brazilian E-Commerce Public Dataset by Olist (Kaggle)

The dataset contains information about:

- Orders
- Customers
- Sellers
- Products
- Payments
- Reviews
- Geolocation data

## Dashboard Structure

### 1. Global Overview

- Total Revenue
- Total Orders
- Average Order Value
- Total Customers
- Delivery Cost Analysis
- Revenue Trends
- Product Category Analysis

### 2. Customer Analysis

- Revenue by Customer Location
- Customer Distribution by State
- Top Customer Cities
- Customer Satisfaction Metrics

### 3. Seller Analysis

- Top Seller Cities by Revenue
- Seller Distribution by State
- Geographic Distribution of Sellers

### 4. Deep Dive Analysis

Interactive page with filters for:

- Year
- Customer State
- Customer City

Allows detailed exploration of revenue, customer behavior, and product category performance.
